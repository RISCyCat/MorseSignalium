# MorseSignalium C++

## Overview

MorseSignalium C++ is the console implementation of the MorseSignalium educational project.

The application can:
- Encode normal text into Morse code.
- Decode Morse code back into readable text.
- Save conversion history locally.
- Demonstrate algorithms, parsing, maps, and string processing in modern C++.

---

## Technologies

- C++17
- STL
- unordered_map
- stringstream
- file handling
- Linux / Windows compatible

---

## Compilation

### Linux

```bash
g++ main.cpp -o MorseSignalium
./MorseSignalium
```

### Windows (MinGW)

```bash
g++ main.cpp -o MorseSignalium.exe
MorseSignalium.exe
```
