# MorseSignalium

## Descriere

**MorseSignalium** este o aplicație educativă și practică pentru codificarea și decodificarea mesajelor în **codul Morse**. 
Proiectul include un GUI simplu, suport audio pentru semnale Morse și algoritmi pentru a traduce text în Morse și invers.

## Funcționalități principale

- Codificare text → Morse
- Decodificare Morse → text
- Redare Morse prin sunet (Windows)
- Suport complet pentru alfabetul A-Z și cifrele 0-9
- Interfață simplă cu Tkinter

## Scurt istoric al codului Morse

Codul Morse a fost inventat de **Samuel Morse** și **Alfred Vail** în secolul XIX, pentru a transmite mesaje rapid prin telegraf. 
Fiecare literă sau cifră este reprezentată printr-o combinație de puncte `.` și linii `-`.

## Cum se utilizează

1. Introdu textul în câmpul **Input**.
2. Apasă **Encode → Morse** pentru a-l transforma în cod Morse.
3. Apasă **Decode ← Morse** pentru a traduce codul Morse înapoi în text.
4. Apasă **Play Morse Audio** pentru a reda mesajul Morse prin sunet.
